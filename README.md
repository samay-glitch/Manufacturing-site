# 🏭 Titan Engineering Solutions

A modern, premium, and fully responsive manufacturing company website built with **React and Vite**.

Titan Engineering Solutions is a professional industrial website designed to showcase manufacturing capabilities, engineering solutions, products, machinery, quality standards, and business services.

The website focuses on creating a strong digital presence for manufacturing companies with a clean industrial design, smooth animations, and a client-focused user experience.

# 🚀 Getting Started

## Prerequisites

Make sure you have installed:

- Node.js
- npm

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/titan-engineering-solutions.git
```

Navigate into the project folder:

```bash
cd titan-engineering-solutions
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```
